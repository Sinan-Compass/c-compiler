# c语言文法  



##  1.程序结构 
```html
<程序>         ::= { <函数定义> }
<函数定义>     ::= <类型说明符> <标识符> '(' [ <参数列表> ] ')' <块>
<参数列表>     ::= <参数> { ',' <参数> }
<参数>         ::= <类型说明符> <标识符>
<类型说明符>   ::= "int" | "float" | "void"
```
## 2.语句块与声明
```html
<块>           ::= '{' { <变量声明> } { <语句> } '}'
<变量声明>     ::= <类型说明符> <标识符> ';'   (* 每行声明一个变量 *)
```
##  3.语句类型
```html
<语句>         ::= <表达式语句>
                | <块>
                | <if语句>
                | <while语句>
                | <return语句>
                
<表达式语句>   ::= <表达式> ';'
<if语句>       ::= "if" '(' <表达式> ')' <语句> [ "else" <语句> ]
<while语句>    ::= "while" '(' <表达式> ')' <语句>
<return语句>   ::= "return" [ <表达式> ] ';'
```
## 4.表达式结构 (优先级从低到高) 
```html
<表达式>       ::= <赋值表达式>
<赋值表达式>   ::= <逻辑或表达式>
                | <标识符> '=' <赋值表达式>   (* 赋值是右结合 *)

<逻辑或表达式>  ::= <逻辑与表达式> { "||" <逻辑与表达式> }
<逻辑与表达式>  ::= <相等表达式> { "&&" <相等表达式> }
<相等表达式>    ::= <关系表达式> { ("==" | "!=") <关系表达式> }
<关系表达式>    ::= <加法表达式> { ('<' | '>' | "<=" | ">=") <加法表达式> }
<加法表达式>    ::= <乘法表达式> { ('+' | '-') <乘法表达式> }
<乘法表达式>    ::= <基本表达式> { ('*' | '/') <基本表达式> }
```

##  5.基本表达式元素 
```html
<基本表达式>    ::= <标识符>        (* 变量引用 *)
                | <整数常量>       (* 整数 *)
                | <浮点数常量>     (* 浮点数 *)
                | '(' <表达式> ')' (* 括号表达式 *)
                | <函数调用>       (* 函数调用 *)

<函数调用>     ::= <标识符> '(' [ <实参列表> ] ')'
<实参列表>     ::= <表达式> { ',' <表达式> }
```
## 6.词法单元 (由词法分析器提供) 
```html
<标识符>       ::= 字母 (字母 | 数字 | '_')*   (* 变量/函数名 *)
<整数常量>     ::= 数字+                       (* 整数 *)
<浮点数常量>   ::= 数字+ '.' 数字+            (* 浮点数 *)

```
